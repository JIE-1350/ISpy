## Installation
### Frontend
#### Requirements
* nodeJS 16.13.0 or newer (LTS)
* npm

#### Setup

```cd frontend/```

```npm install```

#### Run frontend only
```npm start```

#### Run frontend and backend
```npm run dev```

---

### Backend
#### Setup
Create a virtual environment and install the dependencies:

```cd backend/```

```python3 -m venv venv```

```source venv/bin/activate```

```pip install -r requirements.txt```


#### Run manually for testing:

```python server.py```
